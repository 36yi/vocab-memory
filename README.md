# 단어 암기 카드 앱 - 설치 없는 버전

## 실행 방법

`index.html`을 더블클릭해서 브라우저로 열면 됩니다. `npm install` 필요 없습니다.

## 기능

- Day 선택
- 영어 단어 카드 표시
- 카드 클릭 시 뜻 표시
- 스페이스바로 카드 뒤집기
- 좌우 화살표와 Enter로 버튼 조작
- 알았다 / 몰랐다 선택
- 몰랐다 선택 시 브라우저 localStorage에 저장
- 단어별 모른 횟수 기록
- 나중에 볼 단어 목록 확인
- 나중에 볼 단어를 전체 또는 Day별로 다시 학습
- 목록에서 개별 해제
- 나중에 볼 단어만 다시 학습
